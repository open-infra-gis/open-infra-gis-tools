# Public Geospatial Data Examples

This folder is intended for examples based on public, generic, synthetic, or non-sensitive geospatial datasets.

## Planned themes

- Public WMS/WFS services
- Administrative boundary clipping
- Land-cover analysis
- Elevation context
- NASA Black Marble / VIIRS night-time lights
- Disaster impact context
- Environmental monitoring
- Infrastructure exposure analysis

## Safety

Do not add private, restricted, corporate, operational, or personal datasets.
