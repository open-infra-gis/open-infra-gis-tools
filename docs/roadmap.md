# Roadmap

Open Infra GIS Tools is an early-stage open-source GIS toolkit focused on practical, reusable, and non-sensitive workflows for infrastructure-scale geospatial data, environmental monitoring, public datasets, and spatial data quality.

## Guiding principles

- Keep tools small, practical, and easy to adapt
- Use public, generic, or placeholder-only examples
- Avoid sensitive operational data
- Support both GIS analysts and developers
- Prefer clear documentation over complex abstractions
- Treat proxy indicators as supporting context, not direct proof
- Focus on infrastructure resilience, environmental monitoring, and preventive planning

## Phase 1: Professional repository foundation

- README with clear project positioning
- MIT license

- Data safety guidance
- Tool catalog
- Roadmap

- Initial Oracle Spatial, QGIS, OGC, GeoServer, and public-data examples

## Phase 3: GIS workflow quality
